leetcode - Longest Common Prefix

Q:

Write a function to find the longest common prefix string amongst an array of strings.

Code:

我的思路：

从第一个字符开始依次比较每个string，如果均相同则比较下一个，同时计数j加1，直到遇到不同字符。打印出第一个string的前j个字符。
